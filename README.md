## Project setup
* 종속성 설치
```
npm install
```
* Dev Server 실행
```
npm run serve
```
* 빌드
```
npm run build
```
## Project Description
Vue.js를 처음 접해보고,
Vue.js의 기본적인 문법과 Vue.js에 대해 이해하고자 만들어본 프로젝트입니다.<br>
Frontend만 제작되었으며 정상적으로 사용하기 위해서는 전용 API가 필요합니다
